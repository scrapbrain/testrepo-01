The Git Integration for JIRA add-on is for users of the Git source code control system and Atlassian JIRA

The JIRA Git Plugin is an add-on for your existing JIRA server that combines the data in your Git repository with the projects and issues in JIRA.

No special knowledge or experience with Git is needed

The Git Integration for JIRA add-on is simple and easy to use.

Buy now Free trial

Git Code Review

View commits using the links from the Developer Panel.  Review existing to view the git summary. Merged and unmerged branches will be displayed via the Repository Browser.

Git Repository Hosting

Host git repositories on the same machine as JIRA and use JIRA authentication for git.

Git Commits

View who made the commit, time of commit and source code diff via the Git Commits tab on the Issues page.

Git Activity Stream

View list of events and activities by the current user and other people working on the issue.

Supports all Git Servers

Git Integration Plugin for JIRA supports most or all git servers including:

GitLab, Microsoft Team Foundation Server, GitHub, BitBucket, BitBucket Server, Beanstalk, Git on Linux, Git on Windows, Gitolite, Gerrit, GitBlit, Windows share, SSH, HTTP, HTTPS, git protocol, local file path, and network path.

Seamless JIRA and Git integration

Puts Git in context with JIRA to understand development status.

Integrates with JIRA Agile

Git Integration for JIRA add-on supports JIRA Agile.

JQL Search

Search Git commits in JQL with Git Integration for JIRA.

Repository Browser

Browse repositories in JIRA and view files and commits and compare between branches.

Bulk Change

Bulk change for easy management of 1000s of repositories.

Git Summary tab

Git Summary tab summarizes and sorts commit data: first/last commit, days since last commit, lines changed, by developer.

Smart Commits

Add comments, add time, and transition JIRA issues with Git commits.

Commit Email Notifications

The user will get e-mail notifications when watching an issue or repository.

Localization support

English, German, Russian, French, Polish and Japanese language translations.
